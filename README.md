# 📈 Nifty 50 Short-Term Stock Price Prediction System

> 🚀 A machine learning-based system designed to predict short-term stock price movements of Nifty 50 companies using historical data and advanced regression models.

---

## 📌 Overview

Stock market prediction is a challenging task due to its dynamic and non-linear nature. This project aims to build a **short-term stock prediction system** using machine learning techniques to analyze historical price trends and generate accurate forecasts.

The system focuses on **Nifty 50 stocks**, providing insights that can assist traders and investors in making informed decisions.

---

## 🎯 Objectives

- Predict short-term stock price movements  
- Analyze historical trends and patterns  
- Build a high-accuracy predictive model  
- Assist in data-driven investment decisions  

---

## 📂 Dataset

- Source: Historical stock market data (Nifty 50)
- Data includes:
  - Open price  
  - Close price  
  - High price  
  - Low price  
  - Volume  
  - Date/Time  

---

## ⚙️ Tech Stack

- **Language:** Python 🐍  
- **Libraries:**
  - NumPy  
  - Pandas  
  - Scikit-learn  
  - Matplotlib  
  - Seaborn  
- **Environment:** Jupyter Notebook  

---

## 🧠 Machine Learning Approach

### 🔹 Data Preprocessing
- Handling missing values  
- Feature engineering (moving averages, trends)  
- Data normalization  
- Train-test split  

### 🔹 Models Used
- Linear Regression  
- Random Forest Regressor  
- Decision Tree Regressor  

### 🔹 Evaluation Metrics
- R² Score  
- Mean Squared Error (MSE)  
- Root Mean Squared Error (RMSE)  

---

## 📊 Results

| Metric | Value |
|--------|------|
| R² Score | High |
| RMSE | Low |
| Best Model | Random Forest |

✅ The model provides reliable short-term predictions based on historical trends.

---

## 📈 Key Insights

- Market volatility impacts prediction performance
